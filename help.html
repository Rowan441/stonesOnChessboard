Place ordered stones on an infinite square grid. The goal is to place as many stones as possible, starting with a flexible number of 1's, then placing 2, 3, 4, ... in order. A non-1 stone may only be placed on an empty cell whose eight neighbours sum to exactly that stone's value.

Quick rules

  1. You may place any number of 1 stones anywhere on the grid. These are the only stones that may be placed freely.
  2. After you finish placing 1's, you attempt to place 2, then 3, then 4, and so on in strict increasing order. You cannot skip a number.
  3. To place a stone with value a given value, choose an empty cell. The sum of the values of the eight neighbouring cells must equal exactly the value of the placed stone.
  4. Once placed, stones are permanent (unless you use the game's undo). Stones count toward neighbour sums for later placements.
  5. The game ends when you cannot legally place the next integer.

Given n starting stones, the highest value stone able to be placed is known for n < 7. See if you can figure out way to place the maximum number of stones!
